
# Начало работы

Добро пожаловать в документацию Amnezia.

В этом разделе вы узнаете, как создать аккаунт, развернуть свой первый VPN-сервер и подключить устройства.

## Что такое Amnezia

Amnezia позволяет развернуть собственный VPN на арендованном сервере или использовать готовые решения для безопасного доступа в интернет.

### Перед началом

Убедитесь, что у вас есть:

* Аккаунт Amnezia
* Доступ к панели управления
* Компьютер или смартфон для подключения

## Быстрый старт

1. Создайте аккаунт
2. Выберите VPN-решение
3. Разверните сервер
4. Установите приложение
5. Подключитесь
