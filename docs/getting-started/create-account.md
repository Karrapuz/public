# Создание аккаунта

Для использования сервисов Amnezia необходимо зарегистрировать аккаунт.

## Регистрация

1. Откройте страницу регистрации.
2. Укажите адрес электронной почты.
3. Создайте надежный пароль.
4. Подтвердите регистрацию через письмо.

## Требования к паролю

Рекомендуется использовать:

* не менее 12 символов;
* заглавные и строчные буквы;
* цифры;
* специальные символы.

## Подтверждение почты

После регистрации на указанный адрес будет отправлено письмо с ссылкой для активации аккаунта.

Если письмо не пришло:

* проверьте папку Спам;
* убедитесь в правильности адреса;
* запросите повторную отправку.
