# Что такое Amnezia

Amnezia — это платформа для создания и управления собственными VPN-серверами.

В отличие от коммерческих VPN-сервисов, вы используете собственный сервер и самостоятельно контролируете свои данные.

## Преимущества

### Конфиденциальность

Трафик проходит только через ваш сервер.

### Высокая скорость

Отсутствуют ограничения общего VPN-пула.

### Полный контроль

Вы самостоятельно выбираете:

* Локацию сервера
* VPN-протокол
* Количество пользователей
* Настройки безопасности

## Поддерживаемые протоколы

* AmneziaWG
* WireGuard
* OpenVPN
* XRay
* Shadowsocks

## Как это работает

1. Вы арендуете сервер
2. Amnezia устанавливает VPN автоматически
3. Вы подключаете устройства
4. Трафик проходит через ваш сервер
